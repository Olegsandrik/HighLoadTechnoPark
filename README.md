# HighLoadTechnoPark
Репозиторий работы по высоконагруженным системам Технопарка от ВК
# Тип сервиса
Маркетплейс одежды
# MVP
1) Регистрация
2) Поиск товара в каталоге
3) Просмотр товара
4) Покупка товара
5) Выставление товара на продажу
